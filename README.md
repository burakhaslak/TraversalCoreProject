# ✈️ Traversal Reservation - Asp.Net Core 8.0 Project

Traversal is a comprehensive dynamic reservation system built with Asp.Net Core 5.0 using N-Tier (Layered) Architecture. This project covers modern web development patterns, advanced library integrations, and robust backend structures.

---

## 🚀Highlights

### Authentication & Identity
Implemented a full-featured identity system with Custom Identity Validators. Users can register, log in, update profiles with images, and perform "Forgot Password" actions via email integration.

### Advanced Reporting & Operations
- Exporting tour and reservation data to Excel and PDF formats.
- Email sending services for communication.
- CQRS & MediatR patterns were implemented to handle complex data flows efficiently.
- Unit of Work implementation for atomic database transactions.

---
### 🌍 Localization & UI
The project supports Multi-language (Localization), allowing users to switch between different languages. The dashboard is enriched with View Components and Partial Async views for a seamless experience.


---

## 🛠️ Tech Stack & Key Features

- Backend: .NET Core 5.0, C#, Entity Framework Core
- Architecture: N-Tier Architecture (Data Access, Entity, Business, UI, DTO, API)
- Database: MS SQL Server
- Security: Identity Library (Login, Register, Role Management, Password Reset)
- Design Patterns: Repository Pattern, Unit of Work, CQRS, MediatR
- Validation & Mapping: Fluent Validation, AutoMapper
- API & Real-Time: Web API Integration, SignalR (Theory/Implementation)
- Logging & Reporting: Serilog, Excel & PDF Reporting
- UI/UX: Areas (Admin, Member, Default), View Components, AJAX, Multi-language Support (Localization)
